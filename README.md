# odin-projects

A monorepo of frontend projects built while working through [The Odin Project](https://www.theodinproject.com/) curriculum — an open-source full-stack web development course.

All projects are built with **HTML**, **CSS**, and **JavaScript** (no frameworks).

---

## Projects

| Project                                                      | Description                                                                                                                    |
| ------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------ |
| [`admin-dashboard-project`](./admin-dashboard-project)       | An admin dashboard UI built with CSS Grid                                                                                      |
| [`etch-a-sketch-project`](./etch-a-sketch-project)           | An interactive browser-based drawing grid using DOM manipulation                                                               |
| [`form-project`](./form-project)                             | A styled sign-up form with custom CSS and form validation                                                                      |
| [`landing-page-project`](./landing-page-project)             | A fully responsive landing page built with Flexbox                                                                             |

| [`tic-tac-toe-project`](./tic-tac-toe-project)               | A two-player Tic Tac Toe game using modular JavaScript                                                                         |

## Tech Stack

- **HTML5** — semantic structure

- **JavaScript** — DOM manipulation, game logic, object-oriented patterns
- **Webpack 5** — module bundling and asset management
- **npm** — package management

---

## Getting Started

Each project lives in its own subfolder. Most projects run directly in the browser with no build step:

```bash
git clone https://github.com/Feziek/odin-projects.git
cd odin-projects/tic-tac-toe-project
open index.html
```

Projects that use Webpack (e.g. `weather-app`) require a build step:

```bash
cd odin-projects/weather-app
npm install
npm start
```
