# Odin Todo List 

> 🚧 Work in Progress

A vanilla JavaScript single-page application for managing tasks and projects, built with a clean service-component-page architecture.

## Tech Stack

- Vanilla JavaScript (ES6 Modules)
- Webpack 5
- HTML/CSS

## Architecture

- **Services** — handles all data logic and localStorage operations
- **Components** — reusable UI elements that accept callbacks from pages
- **Pages** — orchestrates components, attaches event listeners, and manages rendering

## Features

- [x] Routing between pages
- [x] Add tasks
- [x] Delete tasks
- [ ] Edit tasks
- [ ] Projects
- [ ] Assign tasks to projects
- [ ] Mark tasks as complete

## Status

Currently in active development. Core task CRUD and client-side routing are functional. Projects section coming soon.
